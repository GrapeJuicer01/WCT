## 主界面
![配置数据导入](doc/FunctionDescription/img/04/04-数据导入主界面.png)

## 基本功能说明
把收到的配置数据文件，导入到目标数据库中

## 输入
- 收到的数据文件
    
## 输出 
- 数据库中的数据内容更新
- 界面显示：数据对比结果

## 功能详细说明：
- Import Package
> 导入的数据文件包，该文件包需为使用导出界面导出的数据文件或使用导入界面数据备份功能生成的数据文件。
- Backup Package
> 备份数据的文件名
- Backup Data Firstly
> 如果勾选了，则数据导入前会将数据先导出备份成文件。默认需要先备份。
- Check Table Structure
> 是否需要比对表结构。如果勾选了，则如果源库与目标库的表结构不一致就会报错，导入失败；如果不勾选，则即使表结构不一致，也可以成功导入。
- Sync Table Structure
> 是否需要同步表结构。如果勾选了，则导入数据前会先将目标库中的表drop掉，使用导出时生成的建表语句create表。如果需要同步表结构在导出时就必需同时导出表结构。
- Import By Increment
> 是否为增量导入模式。如果想增量导入，导出时必需为Released By Increment。如果勾选了，则会先将源数据与目标库中的数据进行比较，将比对差异展示出来，确认后导入有差异的数据，相同部分不会导入。差异数据展示界面示例如下：
